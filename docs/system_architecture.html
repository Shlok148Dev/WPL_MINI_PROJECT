Layer 3 — Application Logic (JavaScript Modules)

5 modular JavaScript files handling all client-side business logic

products.js Static product data store
20 items across 4 categories
Exposes window.productsData
main.js Core orchestrator
Featured display, catalogue render
Theme toggle, cart counter, mobile menu
cart.js Cart management engine
Add / Remove / Update qty
Total calculation, checkout flow
search.js Real-time search engine
Keyword filter + category filter
Live results as user types
form-validation.js Input validation & security
Regex checks (email, phone, name, password)
XSS sanitization, password strength meter

→ products.js is loaded first and feeds data into main.js, search.js, and cart.js via window.productsData

↓ ↑ Read / Write Operations (JSON serialization via localStorage API)
Layer 4 — Client-Side Persistent Storage
localStorage Browser-native key-value storage
janka_cart Stores cart items as JSON array
{id, name, price, image, quantity}
janka_theme Persists user's theme preference
"dark" or "light" mode string
▼ Future Backend Integration Layer (Node.js / Express / PHP) ▼
Layer 5 — Database Schema (PostgreSQL)

Relational schema designed for backend integration — currently serves as data blueprint

categories category_id (PK, SERIAL)
category_name VARCHAR(50)
description TEXT
products product_id (PK, SERIAL)
product_name VARCHAR(100)
price DECIMAL(10,2)
category_id (FK → categories)
image_url, stock_quantity
contact_submissions submission_id (PK, SERIAL)
name VARCHAR(100)
email VARCHAR(100)
message TEXT
submission_date TIMESTAMP
user_registrations user_id (PK, SERIAL)
full_name VARCHAR(100)
email VARCHAR(100) UNIQUE
phone VARCHAR(15)

FK Relationship: products.category_id → categories.category_id (One-to-Many)

🌗 Dark / Light Mode Toggle with localStorage persistence
🛡️ XSS Prevention via input sanitization (HTML entity encoding)
📱 Responsive Design — mobile-first with CSS media queries
📦 20 Products across 4 Categories: Rings, Necklaces, Earrings, Bracelets
🔑 Password Strength Meter with real-time visual feedback
⭐ Interactive Star Rating system for feedback
